# MedAgenda

> **"A agenda de compromissos feita para estudantes de Medicina."**

## Objetivo

O MedAgenda é uma aplicação web de agenda de compromissos desenvolvida especificamente para atender às necessidades dos estudantes de Medicina. A rotina médico-acadêmica é complexa e intensa: aulas teóricas, plantões, ambulatórios, laboratórios, provas e congressos se acumulam ao longo da semana. O MedAgenda oferece uma forma simples, rápida e visual de organizar todos esses compromissos em um único lugar, sem a burocracia de ferramentas genéricas.

A proposta não é ser mais um aplicativo de produtividade — é ser a agenda certa, feita para quem vive a rotina da Medicina.
